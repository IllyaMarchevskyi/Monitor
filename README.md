## Встановлення python на WINDOWS
1) Завантажте файл [**install_python.ps1**](https://github.com/IllyaMarchevskyi/Monitor/blob/main/scripts/install_python.ps1) з папки **scripts**
2) В деристорії де знаходиться файл запустіть **PowerShell**
3) Запустіть наступну команду
```
powershell -ExecutionPolicy Bypass -File .\install_python.ps1
```
